# 截图来源说明

这些操作截图来自 OpenAI 官方开发者文档中的真实 Codex 产品截图。为了避免暴露本机账号、项目路径、聊天内容和本地文件，未截取用户本机 Codex Desktop 界面。

真实界面可能会随 Codex 版本、账号权限、操作系统、主题和已安装插件变化。文档中的截图用于帮助读者识别功能位置和界面形态，不代表每个步骤在当前版本中完全一致。
